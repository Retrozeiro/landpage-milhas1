Projeto pessoal

Estúdio digital para compartilhar artes em destaque

Um one-pager leve construído para mostrar peças, contar um pouco do processo criativo e explicar como o projeto é hospedado na Cloudflare Pages.

Hospedado 100% com Cloudflare Pages, pronto para deploy contínuo a partir da pasta public/.

Destaca arte digital +30 trabalhos
Tempo médio de montagem 4 dias
Pronto para mostrar aos clientes Cloudflare Pages

História do projeto

Uma narrativa curta para explicar o propósito e o papel da comunidade criativa por trás das artes.

Processo

Texturas e composições feitas para expressar o ritmo acelerado de trabalhar com milhas e viagens.

Experiência

Design com tipografias fluidas, grid modular e iluminação suave para destacar cada peça.

Resultados

Versões impressas para mostras, combinações para redes sociais e materiais conceituais para eventos.

O motivo desta vitrine

Sem login, sem backend complexo: apenas conteúdo estático que mostra arte e história.

O objetivo é demonstrar clareza visual e atenção ao detalhe, usando apenas HTML, CSS e JavaScript leve. As artes são exemplos fictícios para compor o visual do site, enquanto o cabeçalho reforça a proposta de hospedagem em Cloudflare para quem visita o portfólio.

  • Layout responsivo com foco mobile-first.
  • Textos e CTAs prontos para apresentar a comunidade.
  • Galeria com imagens de exemplo mantidas em fontes públicas.

Login de demonstração

Um formulário local que simula a autenticação e mantém o estado no aparelho, sem conectar a nenhuma API.

Exemplos de artes

Imagens fictícias que ilustram o tipo de conteúdo compatível com o projeto.
